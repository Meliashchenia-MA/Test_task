Для обучения использовался бэкбон ViT-B/16 (модель с использованием трансформеров, предназначенная для классификации изображений). Данная модель была выбрана по причине ее большой мощности и относительно малого размера. У обученной модели точность на тестовой выборке составила около 60%. Данные по сравнению с актуальными моделями находятся в файле comparation.xlsx (данные для сравнения были взяты с сайта https://pytorch.org/vision/main/models.html). Для увеличения точности модели можно предложить следующее: подбор более подходящего бэкбона, более точная настройка гиперпараметров, увеличение количества данных для обучения. Данные об обучении модели представлены в файле Model.ipynb
